# PHP + MySQL Recipe Website

A dynamic recipe website built with PHP and MySQL. The system allows users to view, add,
and manage recipes using server-side logic, form handling, validation, and database storage.

This repository includes the database schema as a `.sql` file for easy setup.

## Technologies
- PHP
- MySQL
- HTML
- CSS

## Features
- Dynamic recipe listing from MySQL
- Add new recipes via forms (PHP POST)
- Server-side validation and error handling
- CRUD operations (Create, Read, Update, Delete)
- Relational database design (tables + relationships)

## Database
The database schema is provided in:
- `database/recipes.sql` (example path)

Typical tables may include:
- `recipes` (recipe details)
- `categories` (recipe categories)
- `users` (if login is implemented)

> Update the table names above to match your real schema.
